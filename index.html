How it works

Three steps. Zero friction.

From npm install to live meeting transcripts in Claude Code. No accounts, no API keys, no configuration.

1

Install

One npm command installs the CLI and macOS menu bar app. Whisper model downloads automatically.

2

Record

Click record in the menu bar or let it auto-detect meetings. Audio stays 100% on your machine.

3

Ask Claude

Transcripts stream to Claude Code via MCP. Ask about your meeting while it is still happening.

Features

Built for developers
who value privacy

No cloud, no subscriptions, no compromises. Everything runs on your hardware.

100% Local

Audio is captured and transcribed entirely on your machine. Nothing is uploaded, ever. Your conversations stay yours.

Zero Cost

Free and open-source. No API keys, no subscriptions, no usage limits. Uses whisper.cpp for on-device speech-to-text.

Invisible

No bot joins your meeting. Meetflow captures system audio directly. Participants never know you are transcribing.

Real-time

Transcripts appear while you are still talking. No waiting for meetings to end. Query your conversation in real-time.

Developer-first

Integrates with your workflow via MCP. Ask Claude Code about your meeting, create tickets, and write code from discussions.

Smart Detection

Automatically detects when meetings start. Labels speakers, filters hallucinations, and produces clean transcripts.

Get started

What happens when you install

The setup script handles everything automatically so you can focus on your meetings, not configuration.

  • ✓ Installs the macOS menu bar app for recording control
  • ✓ Downloads the whisper.cpp model for on-device STT
  • ✓ Configures Claude Code MCP integration automatically
  • ✓ Sets up audio capture for system and microphone
  • ✓ Creates the background transcription service 
# Install globally from npm
npm install -g @meetflow-ai/meetflow

# Run the setup wizard
meetflow-setup
# Start transcribing (or use the menu bar app)
meetflow start
